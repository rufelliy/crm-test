Бізнес-логіка інкапсульована в CallObserver та виконується при створенні дзвінка. 
Автоматично змінюється статус ліда. 
Можливість зміни статусів ліда напряму через API відсутня. 

Пакет ізольований і може бути перевикористаний у будь-якому Laravel-проєкті.

==================================
Для встановлення виконати команду
composer require rufelliy/crm-package
та запустити міграції
php artisan migrate
==================================

Можливі покращення:
- додавання тестів
- додати індекси
- з зростанням логіки, винесення її з обсервера в окремий сервіс
